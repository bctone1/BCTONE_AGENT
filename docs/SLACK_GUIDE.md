# 비씨톤 사용 가이드

GrowFit 팀 전용 Slack AI 비서입니다.
채널 대화를 자동 수집하고, 질문에 답하고, GitHub 변경사항을 요약합니다.

---

## 채널 안내

| 채널 | 용도 |
|------|------|
| `#bctone-전체` | 팀 소통, 진행상황 공유, @비씨톤 질문 |
| `#dev` | 기술 논의, @비씨톤 기술 질문 |
| `#bot-log` | 봇이 자동으로 리포트를 올리는 채널 (읽기 전용) |

---

## 1. @비씨톤 멘션 — 질문하기

`#bctone-전체` 또는 `#dev` 채널에서 `@비씨톤`을 멘션하면 AI가 답변합니다.

### 예시

```
@비씨톤 안녕하세요
```
→ 인사 응답

```
@비씨톤 JWT 토큰 갱신 방식 설명해줘
```
→ 기술 질문 답변

```
@비씨톤 지금 백엔드 진행상황 정리해줘
```
→ 수집된 진행상황 기반 요약

### 알아두면 좋은 것
- 스레드(답글) 안에서도 대화가 이어집니다. 이전 맥락을 기억합니다.
- 단순한 질문은 빠르게, 복잡한 질문(코드 분석, 아키텍처 등)은 더 강력한 AI가 처리합니다.

---

## 2. 슬래시 커맨드 — /bctone

아무 채널에서 `/bctone`을 입력하면 사용할 수 있습니다. 결과는 **나에게만 표시**됩니다.

### 도움말

```
/bctone
```
→ 사용 가능한 명령어 목록

### 팀 진행상황 요약

```
/bctone 요약
```
→ 최근 채널에서 수집된 진행상황과 결정사항을 역할별로 정리해서 보여줍니다.

### GitHub 요약

```
/bctone github
```
→ 백엔드 + 프론트엔드 + 기획 레포 전체 요약

```
/bctone github backend
```
→ 백엔드(GF_Backend) 최근 PR/커밋 요약

```
/bctone github frontend
```
→ 프론트엔드(GF_Frontend) 최근 PR/커밋 요약

```
/bctone github planning
```
→ 기획(mintlify-docs) 최근 변경 요약

### TODO 관리

```
/bctone todo
```
→ 미완료 할일 목록

```
/bctone todo done
```
→ 완료된 할일 목록

```
/bctone todo 완료 #3
```
→ 3번 할일을 완료 처리

## 3. 자동 수집 — 그냥 대화하세요

`#bctone-전체`과 `#dev` 채널에 메시지를 쓰면 비씨톤이 **조용히** 내용을 분석합니다.
봇이 반응하거나 답글을 달지 않습니다. 백그라운드에서 자동 분류만 합니다.

### 어떤 메시지가 수집되나요?

| 이런 메시지를 쓰면 | 비씨톤이 이렇게 분류 |
|-------------------|-------------------|
| "로그인 API 구현 완료했습니다" | **진행상황**으로 저장 (14일 보관) |
| "인증 방식은 JWT로 확정합니다" | **결정사항**으로 저장 (영구 보관) |
| "금요일까지 와이어프레임 완성해야 함" | **TODO**로 저장 (담당자, 기한 자동 추출) |
| "점심 뭐 먹을까요?" | 무시 (수집 안 함) |

### TODO 자동 완료

채널에서 아래처럼 쓰면 해당 TODO가 자동 완료 처리됩니다:

```
완료 #3
```

## 4. 자동 리포트

비씨톤이 `#bot-log` 채널에 자동으로 리포트를 올립니다. **별도 설정 없이 자동 실행**됩니다.

| 리포트 | 시간 | 내용 |
|--------|------|------|
| **일일 리포트** | 평일 오후 6:00 | 오늘 GitHub PR/커밋 + 팀 진행상황 + 미완료 TODO |
| **주간 리포트** | 금요일 오후 6:00 | 이번주 GitHub 종합 + 팀 진행상황 + 미완료 TODO |

리포트에는 백엔드, 프론트엔드, 기획 3개 레포의 활동이 모두 포함됩니다.

## 5. 자주 묻는 질문

**Q: 봇이 내 메시지를 다 읽나요?**
A: 봇이 참여한 채널의 메시지만 분석합니다. DM이나 다른 채널은 접근하지 않습니다.

**Q: 잡담도 저장되나요?**
A: 아닙니다. AI가 업무 관련 내용만 분류해서 저장합니다. 일상 대화는 무시합니다.

**Q: @비씨톤 멘션 안 하면 봇이 대답하나요?**
A: 아닙니다. 멘션 없이 쓴 메시지는 조용히 수집만 하고 응답하지 않습니다.

**Q: 슬래시 커맨드 결과를 다른 사람도 볼 수 있나요?**
A: `/bctone` 결과는 나에게만 표시됩니다. 다른 팀원에게는 보이지 않습니다.

**Q: TODO에 담당자를 지정하려면?**
A: 메시지에 멘션을 포함하면 됩니다. 예: `@damien 금요일까지 API 문서 작성해야 함`
